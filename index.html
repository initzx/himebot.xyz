What is Himebot?

Himebot is a simple yet featureful Discord music bot, that delivers many essential and unique features a music bot must have, while maintaining its simplicity.

Who made Himebot?

Hime is coded in Java using the JDA library by me (init0#8366). It was originally coded in Python as a hobby for me to learn programming, but it has grew a lot since then and is currently used in servers.
